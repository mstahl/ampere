Starting Up Ampere Development Again, After a Long Hiatus

16Nov

My apologies to any Ampere users or potential Ampere users out there who have been turned off by the lack of active development on it. I am now starting to work on a new version of Ampere that will be compatible with the latest MRI and JRuby implementations and also Rails 4. It remains to be seen whether or not this new version will be compatible with older versions of Rails but I will try to make it as compatible as possible. I use semantic versioning for Ampere, so pay attention to version numbers when deciding which version of Ampere to use for your project.

I’d also like to hear from y’all, my userbase. Are there features you’d like to see in Ampere but that aren’t there? Have you experienced any issues or performance issues you’d like to share with the rest of the class? How’s your life in general? You doing okay? We all want to know these things. Let’s rap.

Anyways I’mma get to it now. See y’all later.
